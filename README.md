# Proxecto de fin de ciclo

O obxectivo é desenvolver un software para xestionar a información do catálogo de banda deseñada
(cómics) existente nunha libraría. A libraría conta con diferentes coleccións de banda
deseñada. Ademais, cada colección terá diferentes números. Cada número terá as súas
propias características (data de adquisición, número de colección, tapa,...). Ademais, cada
un deses números estará nun estado determinado (roto, bo estado,...). A aplicación debe
xestionar a información relativa aos recursos existentes na colección, as súas características
e o seu estado.
