Catálogo de colecciones

Esta pantalla se compone de cuatro secciones principales:

  • Barra de menú: en esta barra se encuentran diferentes tipos de utilidades:
    • Informes: permite generar y visualizar informes relacionados con las colecciones.
  • Botón de ver colecciones: carga colecciones de la base de datos en la tabla de la pantalla.
  • Tabla de colecciones central: en esta tabla se mostraran las colecciones de la librería. Además podremos hacer click en una colección para mostrar nuevas opciones:
    • Insertar: abre un diálogo para insertar una nueva colección en la base de datos.
    • Editar: abre un diálogo para editar la colección seleccionada en la tabla.
    • Eliminar: elimina la colección seleccionada en la tabla
